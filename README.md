# Farm PRO - Sistema de Controle de Rebanho

Sistema completo para gerenciamento de fazendas com controle de animais, pesos, vacinas, eventos sanitários e operações comerciais.

## 🚀 Características

- ✅ Dashboard intuitivo com estatísticas
- ✅ Gerenciamento de animais por raça e lote
- ✅ Controle de pesos e comparativos
- ✅ Registro de vacinas e eventos sanitários
- ✅ Análise de dados com gráficos
- ✅ Simulador de lucro
- ✅ Gerenciamento de produtores e clientes
- ✅ Controle de fornecedores
- ✅ Notas de entrada e contratos de venda
- ✅ Operações NF-e
- ✅ Busca avançada
- ✅ Relatórios
- ✅ Sistema de alertas

## 📋 Estrutura

```
farm-pro-html/
├── index.html      # Página principal com todas as seções
├── styles.css      # Estilos CSS profissionais
├── script.js       # Lógica de navegação
└── README.md       # Este arquivo
```

## 🛠️ Tecnologias

- HTML5
- CSS3 (Variáveis CSS, Grid, Flexbox)
- JavaScript Vanilla
- Font Awesome Icons
- Responsive Design

## 🚀 Deploy no Netlify

1. Acesse [Netlify](https://app.netlify.com)
2. Clique em "New site from Git"
3. Selecione o repositório `farm-ear-tag-system-landing`
4. Clique em "Deploy"

O site será automaticamente publicado e atualizado a cada push no GitHub!

## 👨‍💻 Desenvolvimento

Para adicionar novas funcionalidades:

1. Adicione uma nova `<div class="page">` em `index.html`
2. Adicione um novo item de navegação no sidebar
3. Implemente a lógica em `script.js`
4. Estilize em `styles.css`
